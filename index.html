Lightning MAME Frontend

This program is a MAME frontend designed for Linux and especially Ubuntu. It was created to be easy to use, very fast and focus on the essential features.

You can download and install it right now. Currently only Debian/Ubuntu installer is provided.

If you have problems with Lightning MAME Frontend, take a look at the wiki. If your problem is unknown, you can report an issue on the bug tracker.

Lightning MAME Fronted also need people to work on it. If you are a Python programmer that likes open source, do not hesitate to participate on Github.
